# Methodology

This folder articulates the methodological frame under which this work is to be evaluated. The frame is **structural realism**, drawing on Worrall (1989), Ladyman & Ross (2007), Cartwright (1983), and the Duhem–Quine analysis of underdetermination (Duhem 1906; Quine 1951). Each document develops one aspect of the position.

## Why a methodology folder

A typical machine-learning or physics repository does not include a separate folder for methodological positioning. The decision to include one here reflects the structure of the work. The equation in [`../equation/`](../equation/) is derived from three principles, the third of which asserts that perfect dynamical isolation does not occur. A methodology that presupposes isolation, strict Popperian falsificationism, in particular, is in tension with the content the work asserts. The methodology must match the content; otherwise the methodology refutes the content before any experiment is performed.

Putting the methodology in its own folder, with explicit derivations and explicit comparisons to alternative methodological positions, is the same kind of structural move that motivates the rest of the repository's organization: when the methodological frame is load-bearing for how the work should be read, the frame is first-class content.

## What this folder is not

This folder is not a defense of the work against criticisms. The criticisms have not been made yet, and a defensive posture would be premature. The folder articulates a positive position: the position is structural realism; the criteria are the six listed above; the work is evaluated by those criteria.

This folder is also not a philosophical treatise. The position is articulated as briefly as is consistent with making the argument clear. Readers who want the full philosophical literature can follow the references; the documents here use the philosophical literature to anchor the methodological frame and proceed.
