# Paths

> **Triad** — the minimal equation for persistent entities: state, self-action / memory, and coupling / dissipation.

Five entry routes through the same body of work, each calibrated to where you start. Switching mid-journey is encouraged.

| Path | For readers who arrive |
|---|---|
| [`if-you-are-new.md`](if-you-are-new.md) | With no specific background — curious. |
| [`if-you-are-from-physics.md`](if-you-are-from-physics.md) | Familiar with nonlinear dynamics, field theory, statistical mechanics. |
| [`if-you-are-from-ml.md`](if-you-are-from-ml.md) | Familiar with sequence models, attention, deep learning. |
| [`if-you-are-from-neuroscience.md`](if-you-are-from-neuroscience.md) | Familiar with neural oscillations, gamma rhythms, EEG. |
| [`if-you-are-from-philosophy.md`](if-you-are-from-philosophy.md) | Familiar with structural realism, philosophy of science. |

The paths thread through the same content in different orders. The derivation is the derivation, the methodology is the methodology — paths adjust *order* and *framing*, not the underlying content.
