# Triad Equilibrium Hero

This result documents the rebuilt 3D physical anti-collapse/equilibrium simulation used in the repository README.

## Setup

The simulation compares two runs from the same initial field and the same attractive nonlinear regime:

| Run | Memory | Outcome |
|---|---|---|
| No-memory field | $V_{\text{mem}}=0$ | localized collapse-like concentration |
| Memory field | $V_{\text{mem}}=\sum_j\lambda_jy_j$ | bounded distributed equilibrium |

The solver is `implementation/physics/balanced_solver_3d.py`.

The visual is generated by `experiments/physics/rebuild_equilibrium_hero.py`.

## Command

```bash
source scripts/cuda-env.sh
python experiments/physics/rebuild_equilibrium_hero.py \
  --N 128 \
  --steps 2400 \
  --frame-every 20 \
  --sample-every 10 \
  --fps 18
```

## Reading

The important result is equilibrium, not visual drama.

The no-memory run concentrates density into a localized singular-like state. The memory run passes through collapse pressure but does not stay there. Hierarchical memory changes the attractor: the field releases into an extended bounded state.

In Triad terms, anti-collapse is the mechanism. Bounded distributed equilibrium is the result.
