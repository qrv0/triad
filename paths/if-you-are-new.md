# Start here. No prerequisites.

This work takes one equation — derived from three plain observations about how things last — and shows that the same equation appears, independently, in physics, neuroscience, machine learning, ecology, and beyond. Curiosity is the only prerequisite.

<div class="path-eq" markdown>
$$
i\hbar\, \partial_t \Psi
\;=\;
\left[\,-\tfrac{\hbar^{2}}{2m}\nabla^{2} + V_{\text{ext}} + \Lambda |\Psi|^{2} + V_{\text{mem}} + \alpha\,(-\Delta)^{\sigma/2} - i\Gamma\,\right]\Psi \;+\; \eta
$$

## What this maps to in your area

Think of sand on a struck drum: the vibration organizes the sand into a definite geometric pattern — not randomly, but because of the drum's shape and how it was hit. This work is about an equation that does the same thing in three-dimensional fields. From a featureless starting state it produces a specific crystalline geometry, a 40 Hz-range absorption band that overlaps neural gamma rhythms, and a mechanism that prevents collapse — the same mechanism documented in a 70-million-parameter language model mid-training. The geometry is not put in; it emerges. That recurrence across substrates is the central finding.
